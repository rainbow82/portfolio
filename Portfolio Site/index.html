Beached Animal Rescue
Beached Animal Rescue is an Android app that guides users through what to do if they find a stranded sea mammal. The user's location is displayed using a location API, allowing them to give rescue workers an exact location of a stranded whale, dolphin, etc. Another screen displays information on what the user should and should not do to help the animal while waiting on rescue workers.
